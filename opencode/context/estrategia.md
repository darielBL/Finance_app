# Estrategia de Trabajo

## Perfil del Desarrollador
- Conocimiento sólido de Ruby on Rails (trabaja con ello a diario)
- Usa RubyMine como IDE
- Busca independizarse económicamente mediante apps pequeñas con anuncios
- Trabajo a tiempo completo, necesita maximizar eficiencia

## Filosofía de Trabajo
- Tú (IA) eres el cerebro: decisiones importantes, estrategia, arquitectura
- Claude Code (o el asistente) es el ejecutor para tareas mecánicas
- No se busca que la IA haga toda la app, solo que acelere el desarrollo
- Control y validación humana por encima de automatización ciega

## Reglas Acordadas
Antes de escribir código, debe explicar:
- Qué archivos va a modificar o crear
- Por qué ese enfoque
- Alternativas consideradas (si aplica)
- Esperar confirmación antes de proceder

## Estrategia para Ahorrar Tokens
- Conversaciones cortas y específicas
- Reiniciar contexto por cada módulo o tarea
- Pedir planes antes que código
- Usar archivos de contexto para reglas fijas

## Objetivo Final
- Apps pequeñas (finanzas personales, hábitos, herramientas simples)
- Monetización mediante anuncios como primera vía de ingreso
